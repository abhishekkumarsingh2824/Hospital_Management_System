# Hospital Management System (C++)

A console-based Hospital Management System developed using **C++** and **Object-Oriented Programming (OOP)** concepts.
The system allows efficient management of patients, doctors, and appointment scheduling through a menu-driven interface.

---

## 🚀 Features

* Add Patient Records
* Add Doctor Records
* Schedule Appointments
* View Patient Details
* View Doctor Details
* View Appointment Information
* Unique ID generation for Patients and Doctors

---

## 🛠️ Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL (Standard Template Library – Vector)
* CLI (Command Line Interface)
